# ETL_Process_Employee-Data-Analysis-Visualization
This project demonstrates an end-to-end ETL pipeline (Extract, Transform, Load) combined with Exploratory Data Analysis (EDA) and data visualization using Python.

The objective is to simulate real-world employee data processing and extract meaningful business insights using industry-standard tools.

Problem Statement

Organizations need to understand employee distribution, salary trends, and performance metrics to make data-driven decisions.

This project answers key questions such as:

How does salary vary across departments?
What is the relationship between experience and salary?
Which cities and departments have the highest workforce concentration?
Who are the high-performing employees?
